# ThinkStorm
## サービス概要
AIと協働してアイデアを創出・整理・評価できるブレインストーミング支援アプリです。<br>
個人でもチームでも使える、発散から収束までのプロセス全体を支援するツールです。

#### 主な特徴 :
* 課題を入力するとAIが多角的な視点からアイデアを提
* AIが類似アイデアを自動でグルーピング・カテゴリ分けし大量のアイデアを構造的に整理

## このサービスへの思い・作りたい理由
前職のデータ分析コンサルティング業務では、クライアントと一緒に課題解決のためのアイデア出しや仮説構築を行う場面が多くありました。<br>
その際、ホワイトボードやふせんを使ってブレストをするものの、以下のような課題を感じていました。<br>
* 良いアイデアが出ても、整理や評価が属人的になりがち
* 後日「あの時出たアイデア、何だっけ?」と振り返れない
* 一人でブレストする時に視点が偏りやすい
* アイデアの評価基準が曖昧で、感覚的な判断になってしまう
これらの課題に対して、「AIが能動的にアイデア創出をサポートし、かつそのプロセスを構造的に記録・管理できるツール」があれば、<br>
個人でもチームでもより質の高いアイデア創出ができると考え、このサービスを作りたいと思いました。

## ユーザー層について
### メインターゲット : 一人でアイデア出し・整理をする人
#### 具体例 :
* 起業家、新規事業担当者、企画職、マーケター、プロダクトマネージャー、個人開発者など
#### 選定理由 :
* 一人でブレストすると視点が偏りやすく行き詰まるが、AIが多角的な視点から大量にアイデアを提案することで、発想の幅が広がる
* 生成された大量のアイデアを一人で整理・分類するのは負担が大きいが、AIクラスタリングにより短時間で構造化できる
* アイデア検討のプロセスが記録されるため、上司・チーム・投資家・採用担当者など他者に対して「なぜこのアイデアを選んだのか」を論理的に説明できる

### サブターゲット : チームでアイデア出し・整理をする人
#### 具体例
* 企画会議、ブレストセッション、プロジェクトチームなど（3-10人程度）
#### 選定理由 :
* 会議前にAIで大量のアイデアを生成しておくことで、当日は議論の質を高めることに集中できる
* リアルタイム共同編集により、オンライン会議でも全員が同時にアイデアを追加・移動でき、活発な議論ができる
* 評価・投票機能により、チーム全体の意見を可視化し、納得感のある意思決定ができる

## サービスの利用イメージ
### 典型的な利用フロー
1. 課題の設定: ユーザーが解決したい課題やテーマを入力（例: 「20代向けの新しい健康管理サービスのアイデア」）
2. AIアイデア生成（発散フェーズ）: 課題やテーマからAIが15〜20個のアイデアを提案。ユーザーも自由にアイデアを追加可能。
3. 手動整理: 生成されたアイデアを現状に合わせて編集・削除などで整理
4. AIクラスタリング（収束フェーズ）: ボタンクリックで、AIが類似するアイデアを自動グルーピングし、各グループに適切なカテゴリ名を付与。ユーザーはグループ名の編集、アイデアの移動、新しいグループ作成など自由に調整可能
5. 評価・投票: 各グループまたは個別アイデアをユーザーが設定した評価軸で評価し、スコアを可視化
6. 深掘り対話: 気になるアイデアについてAIと対話しながら詳細を詰める
7. 意思決定: 評価とチーム投票を元に最終的なアイデアを選定

### 想定される利用シーン
* 通勤中のスキマ時間：テーマを入力し、AIが生成した15個のアイデアをざっと眺める。気になるものにメモして、詳細は後で。
* 週末にじっくり考える：AIで20個生成し、クラスタリングで整理。全体を俯瞰することで方向性が見えてくる。
* チーム会議の効率化：会議前にメンバーで事前にアイデア追加。当日は30個から議論開始し、リアルタイムで整理・評価・意思決定。

## ユーザーの獲得について
* Xでの発信：アプリ使用のイメージ・活用シーンやTipsを投稿

## サービスの差別化ポイント・推しポイント
### 独自の提供価値
1. 発散から収束まで一気通貫：アイデア生成(発散)だけでなく、AIクラスタリングによる整理・構造化(収束)まで自動化。ブレストのプロセス全体を一つのツールで完結
2. AIが提案、人間が最終調整：AIが自動で大量に提案・整理しつつ、グループ名やアイデアの配置はユーザーが自由に編集可能。完全自動でも完全手動でもない、バランスの取れた協働
3. 軽量で柔軟な設計：簡単なアイデア出しから、しっかりとしたプラニングまで。一人でもチームでも、自分のペースとスタイルで利用できる

### 競合との比較
| 要素 | 既存ホワイトボードツール | 既存AIチャット | ThinkStorm |
|------|---------------------|--------------|-----------|
| アイデア生成 | 手動入力 | 指示が必要 | **自動提案** |
| 整理・構造化 | 手動で配置 | なし | **AI自動クラスタリング** |
| プロセス記録 | 残らない | 残るが整理されない | **構造化して記録** |

## 機能候補
### MVPリリース時に作っていたいもの
個人利用のための機能開発を想定
#### 機能一覧
* 認証・基本機能
  - ユーザー登録・ログイン（Devise）
  - セッション作成・一覧・詳細・削除
  - 基本的なレスポンシブデザイン
* アイデア発散フェーズ
  - 課題/テーマ入力フォーム
  - AIアイデア生成機能
  - ユーザーによる手動アイデア追加
  - アイデアの編集・削除
* アイデア収束フェーズ
  - AIクラスタリング実行ボタン
  - 類似アイデアの自動グルーピング
  - 各グループへのカテゴリ名自動付与
  - グループごとのカード表示
  - グループ名の編集
  - アイデアのグループ間移動（ドロップダウン選択式）
  - 新しいグループの手動作成

#### MVPのゴール
* 「課題入力 → AI生成（発散） → AIクラスタリング（収束） → 手動調整」というサービスの核心的な価値を検証

### 本リリース
#### 機能一覧
* クラスタリング機能
  - ドラッグ&ドロップでのアイデア移動
  - グループの削除
  - 未分類アイデアの表示
  - クラスタリングのやり直し機能
  
* 評価・分析機能
  - ユーザーが設定可能な評価軸の作成
  - アイデア/グループ単位での評価機能
  - 評価結果の可視化（グラフ/チャート）
  - 評価スコアでのソート・フィルタ
  - AI深掘り対話機能（アイデアについて詳しく聞ける）
  - エクスポート機能（PDF/Markdown）

* ユーザー体験向上
  - ユーザー設定ページ（プロフィール編集、メールアドレス変更、パスワード変更）
  - セッション検索機能
  - アイデアへの詳細なメモ・タグ機能

* チーム機能
  - セッションの共有リンク生成
  - 招待機能（URL共有）
  - リアルタイム共同編集（Action Cable）
  - メンバー一覧表示
  - 誰が追加したアイデアかの表示
  - チーム投票機能
  - 投票結果の集計・可視化
  - コメント機能

## 使用する技術スタック(暫定)
* フロントエンド : TailwindCSS/Hotwire (Turbo + Stimulus)
* バックエンド : Ruby on Rails 8.1.1/Ruby 3.3.6
* データベース : PostgreSQL
* 認証 : Devise
* AI連携 : OpenAI API
* デプロイ : Render
